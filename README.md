# MERN Movie App ([visit](https://tmdb-frontend-chi.vercel.app/))

## Running Locally

```bash
mkdir client && cd client
git clone https://github.com/saadfrhan/tmdb-frontend ./
pnpm install

# .env file
VITE_TMDB_READ_ACCESS_TOKEN=""
VITE_CLERK_PUBLISHABLE_KEY=""
VITE_TMDB_API_KEY=""

pnpm dev
```
